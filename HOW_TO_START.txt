PROBLEM: "localhost refused to connect" or "ERR_CONNECTION_REFUSED"
SOLUTION: The server is not running! You need to start it first.

METHOD 1: EASIEST - Double-Click Batch File
================================================================================

1. Go to this folder:
   "c:\Users\monik\OneDrive\Desktop\INDUSTRY PROJECT\MINOR project codes\data"
